# Banco de Dados

Este repositório contém scripts e projetos relacionados a banco de dados SQL.

## Estrutura

- **clinica-sql/** - Scripts SQL para sistema de gerenciamento de clínica
- **eleicao-sql/** - Scripts SQL para sistema de gerenciamento de eleições

## Tecnologias

- SQL
- Modelagem de Dados
- Banco de Dados Relacionais

## Como Usar

Execute os scripts SQL em seu SGBD (Sistema de Gerenciamento de Banco de Dados) preferido.
